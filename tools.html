VS Code とは？

コードを書くための「メモ帳の超進化版」

📝
💡
一言で言うと
VS Code は プログラムを書くための高機能テキストエディタ です。
Windowsの「メモ帳」と同じ「文字を書くソフト」ですが、プログラマー向けに作られた超進化版です。

01メモ帳と何が違うの？

📄 普通のメモ帳
  • 文字を書くだけ
  • 間違いを教えてくれない
  • どこが何の命令かわからない
  • 保存してファイル実行が必要
  • 追加機能なし
→
📝 VS Code
  • ✓ 色分けでコードが読みやすい
  • ✓ ミスを赤波線で教えてくれる
  • ✓ 入力途中に候補を表示（補完）
  • ✓ 中でターミナルを開いて実行できる
  • ✓ 拡張機能で機能を追加できる

02画面の各部分の名前

VS Code の画面構成
サイドバー
ファイル一覧・検索・Git管理などのアイコンが縦に並ぶ
エディタ
ここにコードを書く（メインの作業スペース）
ターミナル
コマンドを入力してプログラムを実行する場所

03特に便利な機能3つ

1
🎨
シンタックスハイライト
「命令」「文字列」「数字」など、コードの種類によって色が変わります。これにより、コードの構造が一目でわかるようになります。
print("Hello", 42)
2
🔮
コード補完（IntelliSense）
文字を入力すると、続きの候補がリストで表示されます。タイプミスを減らし、コマンドを全部覚えなくていいので初心者に特に便利。
print ▶ print()
3
🐛
エラー表示（Error Lens）
間違いのある行に赤い波線と説明が表示されます。拡張機能を入れると行末に直接エラー内容が出て、すぐ気づけます。
prnt("Hello") ← エラー: prnt は未定義

04VS Code の開き方

Windows
スタートメニュー → 「Visual Studio Code」で検索 → クリック
Mac
Finder → アプリケーション →「Visual Studio Code」をダブルクリック
💡 コツ
フォルダを右クリックして「Code で開く」を選ぶと、そのフォルダをプロジェクトとして開けます
TERMINAL

ターミナルとは？

文字を入力してコンピュータを操作する「コックピット」

⬛
💡
一言で言うと
ターミナルとは 文字を入力してコンピュータに命令を出すウィンドウ です。
クリックやドラッグではなく「コマンド（命令文）」を入力して操作します。

01なぜターミナルが必要なの？

🚀
Pythonを実行するため
書いたプログラムを動かすには python ファイル名.py と入力します。これはGUI（クリック操作）ではできません。
📦
ライブラリをインストールするため
pip install ライブラリ名 というコマンドで便利な部品を追加できます。
⚡
素早く作業するため
ファイルのコピーや移動など、クリックより速くできる操作が多いです。

02ターミナルを使ってみよう

ターミナル
$ python --version
Python 3.12.0
$ python hello.py
Hello, World!
こんにちは！
$ _
$ （ドル記号） 「あなたが入力する場所」の目印。自分で入力する必要はありません。
_ （カーソル） 現在入力できる位置を示します。点滅していることが多いです。
白い文字（出力） コマンドを実行した結果（プログラムの出力）です。自分では入力しません。

03よく使う基本コマンド

コマンド 意味 例
python ファイル名 Pythonを実行する python hello.py
cd フォルダ名 フォルダを移動する cd Documents
ls（Mac） / dir（Win） ファイル一覧を表示 ls
pip install 名前 ライブラリを追加する pip install requests
clear（Mac） / cls（Win） 画面をきれいにする clear

04ターミナルの種類（早見表）

🪟 Windows
コマンドプロンプト（cmd） PowerShell Windows Terminal
初心者は コマンドプロンプト か VS Code内のターミナル がおすすめ
🍎 Mac
ターミナル（Terminal.app） iTerm2（高機能版）
初心者は ターミナル か VS Code内のターミナル がおすすめ
WINDOWS

コマンドプロンプト

Windowsに昔からある黒い画面の正体

🪟
💡
一言で言うと
コマンドプロンプト（cmd）は Windows に昔からある標準のターミナル です。
黒い背景に白い文字が特徴で、「あの黒い画面」と言われるものです。

01どこにある？開き方

方法①
Win + R キーを同時押し → 「ファイル名を指定して実行」が開く → cmd と入力 → Enter
方法②
スタートメニューを右クリック → 「ターミナル」または「コマンドプロンプト」を選択
方法③
エクスプローラーでフォルダを開き、アドレスバーに cmd と入力 → そのフォルダでcmdが開く（便利！）

02画面を見てみよう

コマンドプロンプト ─ □ ×
C:\Users\田中> python --version
Python 3.12.0
C:\Users\田中> python hello.py
Hello, World!
C:\Users\田中> _
C:\Users\田中> 「今いる場所（フォルダのパス）」が表示されています。Cドライブ → Users → 自分のフォルダにいることがわかります。
>（大なり記号） 入力待ちの目印。Macのターミナルの「$」と同じ役割です。

03cmd でよく使うコマンド

コマンド やること 例
python ファイル名.py Pythonを実行する python hello.py
cd フォルダ名 フォルダに移動する cd Desktop
cd .. ひとつ上のフォルダに戻る cd ..
dir ファイル・フォルダ一覧を表示 dir
cls 画面の文字を消す cls
exit ウィンドウを閉じる exit

04よくある困りごとと解決策

❓ python と入力したら「コマンドが見つかりません」と出た
→ Pythonインストール時に 「Add Python to PATH」 にチェックを入れ忘れた可能性があります。Pythonを再インストールしてチェックを入れましょう。
❓ ファイルが実行できない（「見つかりません」エラー）
→ cd コマンドでファイルがある フォルダに移動してから 実行してみてください。dir でファイルの存在を確認できます。
❓ 途中で止まったプログラムを終わらせたい
→ Ctrl + C を押すと強制終了できます。覚えておくと便利！
WINDOWS

PowerShell

コマンドプロンプトの進化版・青い画面の正体

💙
💡
一言で言うと
PowerShellは コマンドプロンプトをパワーアップさせたWindows標準のターミナル です。
青い（または紺の）背景が特徴で、より高度な操作ができます。

01コマンドプロンプトと何が違うの？

⬛ コマンドプロンプト（cmd）
  • Windowsに昔からある
  • 基本的な操作のみ
  • コマンドが少なめ
  • シンプルで覚えやすい
  • Pythonの入門に十分
→ 進化
💙 PowerShell
  • ✓ Windows 10/11で標準になった
  • ✓ より多くのコマンドが使える
  • ✓ プログラミング的な記述ができる
  • ✓ スクリプト（自動化）が書けるし
  • ✓ Macのターミナルに近いコマンドも使える

02PowerShell の開き方

方法①
Win + X キーを同時押し → 「ターミナル」を選択（Windows 11では標準でPowerShellが開く）
方法②
スタートメニューで「PowerShell」と検索 → クリック
💡 おすすめ
VS Code を開いてターミナルパネルを使うのが一番便利！上のメニュー「ターミナル」→「新しいターミナル」で開けます。

03PowerShell の画面

Windows PowerShell ─ □ ×
PS C:\Users\田中> python --version
Python 3.12.0
PS C:\Users\田中> python hello.py
Hello, World!
PS C:\Users\田中> _
PS C:\Users\田中> 「PS」がPowerShellの目印です。あとはコマンドプロンプトとほぼ同じ見た目です。

04初心者がPowerShellで使うコマンド

コマンドプロンプトのコマンドはほぼそのまま使えます。追加で覚えると便利なものを紹介します。

コマンド やること メモ
ls ファイル一覧（cmdの dirと同じ） Macと同じ書き方！
cd フォルダ移動（cmdと同じ） —
Clear-Host / clear 画面を消す cls も使える
python hello.py Pythonを実行（cmdと同じ） —

05⚠ PowerShell でよくあるエラー

❓ 「実行ポリシー」エラーが出てスクリプトが動かない
→ Windows の安全機能です。以下のコマンドを 管理者として実行 したPowerShellで入力することで解除できます：
Set-ExecutionPolicy RemoteSigned
❓ cmdで動いたのにPowerShellで動かない
→ 一部のコマンドはcmdとPowerShellで書き方が違います。Pythonの実行に関しては基本的に同じです。

02場面別おすすめ

🐍 Python のプログラムを実行したい
→ VS Code のターミナル に python ファイル名.py と入力
📦 pip でライブラリを追加したい
→ VS Code のターミナル に pip install ライブラリ名 と入力
📁 フォルダの中身を確認したい
→ VS Code のターミナル で ls（Mac）または dir（Windows）
🆚 どのバージョンか確認したい
→ どのターミナルでも python --version で確認できます

03VS Code 内ターミナルの開き方

手順①
VS Code を開く
手順②
上のメニューバーから 「ターミナル」 をクリック → 「新しいターミナル」 を選択
手順③
画面下半分にターミナルが開く。あとは python ファイル名.py を入力するだけ！
ショートカット
Ctrl + `（バッククォート）キーでターミナルのON/OFFを切り替えられます。Macも同じです。
